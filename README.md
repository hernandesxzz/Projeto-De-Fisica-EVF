# Black Holes: Where Physics Breaks

An interactive, educational website about black holes designed for high school students. Features a futuristic, gamer-inspired UI with smooth animations, engaging transitions, and visually impressive effects.

## Features

- **Futuristic Design**: Space-themed UI with neon colors, dark backgrounds, and tech-inspired typography
- **Smooth Animations**: Powered by GSAP for scroll-triggered animations and transitions
- **Interactive Elements**: 
  - Toggle between Newton's and Einstein's gravity theories
  - Animated black hole formation sequence
  - Visual distortion effects near event horizon
  - Spaghettification animation
  - Interactive cards for black hole types and detection methods
- **Responsive Design**: Works on desktop and mobile devices
- **Visual Effects**:
  - Starfield background with parallax scrolling
  - Glowing accretion disk around black hole
  - Particle effects and subtle glitch animations
  - Cursor follower with space-themed design
- **Educational Content**:
  - Clear explanations of black hole physics
  - Historical context (Newton vs Einstein)
  - Formation process
  - Detection methods
  - The first-ever image (M87*)
  - Curiosities and theoretical concepts (time dilation, wormholes, etc.)

## Technologies Used

- HTML5
- CSS3 (with CSS variables and animations)
- JavaScript (ES6+)
- [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform) for animations
- Google Fonts (Orbitron and Rajdhani for futuristic typography)

## How to Use

1. Clone or download this repository
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)
3. Explore the sections by scrolling or using the navigation
4. Enjoy the interactive learning experience!

## Customization

- **Colors**: Modify the CSS variables in `:root` in `styles.css`
- **Fonts**: Change the Google Fonts links in `index.html` and update font-family in CSS
- **Content**: Edit the text in `index.html` to update information
- **Animations**: Adjust timing and effects in `styles.css` (animation properties) and `main.js` (GSAP timelines)
